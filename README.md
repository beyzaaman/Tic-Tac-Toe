# Tic Tac Toe Project

This project aims to represent a simple text-base version of the Tic Tac Toe game. <br /> <br />
 Checkout for the learn how to play Tic Tac Toe [in here](https://www.wikihow.com/Play-Tic-Tac-Toe).

## Contents

  Initially, Player 1 is assigned as X, Player 2 is O. Project includes three different cases: <br /><br />
  • In first case, two users plays the game with each other. You can reach this option via activating play() function. <br /><br />
  • In second case, user plays with the computer; computer makes it moves randomly. You can reach this option via activating computer_play() function. <br /> <br />
  • In third case, is similar with the second case but right now computer can plays more strategically, it can detect your winning situations and it can block them. You can reach this option via activating smart_computer_play() function.<br />
